Quickstart Guide

Deploy the bot to your server in less than 5 minutes using these optimized steps.

📦

1. Install Node

Ensure you have Node.js 18.0.0 or higher. Version 20 LTS is recommended for security.

🧬

2. Dependencies

Run npm install to fetch the required Discord.js v14 libraries and dotenv.

🔑

3. Secrets

Create a .env file and populate it with your Token and Client ID.

Terminal
git clone https://github.com/ts_122/sold.git
cd sold
npm install
node index.js

JSON Database Manual

The bot utilizes flat-file JSON databases for zero-latency lookups and maximum portability between hosting providers.

config.json
{
  "welcome": { "enabled": true, "channelId": "...", "color": "#HEX" },
  "modmail": { "guildId": "...", "staffRoleId": "...", "categoryId": "..." }
}
Property Type Utility Description
modmail.guildId Snowflake The exact server ID where ticket channels will be created.
modmail.staffRoleId Snowflake The role allowed to view, claim, and reply to ModMail tickets.
modmail.categoryId Snowflake The category ID where incoming ModMail channels will be nested.
welcome.enabled Boolean Global toggle for the automated welcome system.

Comprehensive Hosting Guide

Professional VPS Setup (Linux)

For the most stable hosting environment, use Ubuntu 22.04 LTS and the PM2 process manager.

Bash (Ubuntu)
# Install Node 20
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# Install PM2 Global
npm install pm2 -g

# Start Bot
pm2 start index.js --name "pickle-infra"
pm2 save

Home Server (Windows)

Ideal for low-cost setups. Use a .bat file to manage the process.

run.bat
:loop
node index.js
goto loop

Technical FAQ & Recovery

Slash commands not showing?

Invite the bot via the link in the terminal. Discord takes time to sync global commands; re-inviting forces a fast refresh.

"License Verification Error"

The host machine blocked the outgoing HTTPS request to GitHub. Disable local firewalls or anti-virus during verification.

Failed to DM user?

The user has DMs disabled in their Privacy settings for this server. This is a Discord limitation.
