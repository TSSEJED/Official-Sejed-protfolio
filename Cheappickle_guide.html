Getting Started

Minimum Requirements

  • Node.js 20.0.0 (LTS) or higher
  • NPM 9.0+
  • Operating System: Linux (Ubuntu/Debian) or Windows 10/11

Environment Secrets (.env)

Create a .env file in your root folder with these exact keys:

.env DISCORD_TOKEN=your_bot_token
CLIENT_ID=your_application_id
ORDER_LOG_CHANNEL_ID=log_channel_id

Administrator Commands

Command Subcommand Options Description
/setup welcome channel, message, color, enabled Configures the join/leave system.
/setup modmail guild_id, category, log_channel, staff_role Initializes the support ticket system.
/setup giveaway color Sets the aesthetic theme for prizes.

Support & Ticket Commands

Command Options Role Required Description
/modmail reply message, anonymous Staff Role DMs the user through the ticket.
!reply message Staff Role Quick-reply (no slash required).
/modmail close reason Staff Role Closes ticket & generates transcript.
/modmail claim - Staff Role Assigns the ticket to yourself.
/modmail blacklist user Alt Admin Bans a user from using support.

Raw Gateway Fallback

How It Works

Most Discord bots fail because the library (discord.js) discards DM packets when the "Message Content Intent" is slow to sync. We solved this with a Low-Level Packet Interceptor.

index.js [Internal] client.on('raw', packet => {
  if (packet.t === 'MESSAGE_CREATE') {
    // Manual packet extraction logic...
    // Bypasses event filter...
  }
});
Technical Benefit: 100% Message Delivery. Even if the user is not cached, the bot fetches the profile manually from the API.

Linux VPS (Ubuntu / Debian) Guide

This is the Recommended way to host for 24/7 uptime.

1. System Prep

sudo apt update && sudo apt upgrade -y
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs

2. Bot Setup

git clone {your_repo_link} bot
cd bot
npm install

3. Process Management (PM2)

PM2 keeps your bot alive if it crashes or the server reboots.

npm install pm2 -g
pm2 start index.js --name "Cheappickle"
pm2 save
pm2 startup

Windows Hosting Guide

Suitable for Home Servers or Windows Dedicated Servers (RDP).

  1. Download & Install Node.js v20+ Windows Installer.
  2. Open PowerShell or CMD in the bot folder.
  3. Run npm install.
  4. Launch with node index.js.
Tip: Create a run.bat file with the content node index.js --pause to easily restart.

Game Panel Hosting (Pterodactyl)

If you are using a hosting service like Wisp or Pterodactyl:

  • Node.js Version: Select the Node.js 20 (or latest) egg.
  • Start Command: Ensure it points to node index.js.
  • Files: Upload all files except node_modules and run the panel's "Install" task.
  • Env: Most panels have a "Startup" tab where you can put your token.

FAQ: Expert Troubleshooting

"Cannot read property 'id' of null"

Occurs if the config.json is missing the modmail object. Run /setup modmail immediately after startup.

"User is not receiving replies"

The user may have their DMs turned OFF for the server. The bot will log "Failed to DM user" in the console.

"Rate Limit (429)"

Discord has blocked your IP. If hosting on a VPS, try a different data center or wait 15 minutes.

"License Error"

Ensure the machine can reach GitHub Raw. Firewalls or corporate proxies are the usual cause.
